접두사 : ~

on : 봇을 통화방에 초대한다.
off : 통화방에서 못을 내보낸다.

sh : 검색 목록 중 상위 5개의 결과를 보여준다.
ex) ~sh 피아노맨 네소식

pu : URL을 통해 음악을 재생한다.
ex) ~on https://www.youtube.com/~~

ps : 검색을 통해 음악을 재생 또는 예약한다.
ex) ~ps 산들 취기를 빌려

ps ~~ -s N : 음악 검색 결과 중 N 번째 결과를 재생 또는 예약한다.
ex) ~ps 산들 취기를 빌려 -s 3

ls : 예약 리스트를 확인한다.

sh : 예약 리스트를 셔플하여 순서를 섞어준다.

p : 일시 정지
rs : 일시 정지된 음악을 다시 실행한다.
