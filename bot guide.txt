접두사 : ~

on : 봇을 통화방에 초대한다.
off : 통화방에서 못을 내보낸다.

sh : 검색 목록 중 상위 5개의 결과를 보여준다.
ex) ~sh 피아노맨 네소식

ps : 검색을 통해 음악을 재생 또는 예약한다.
ex) ~ps 산들 취기를 빌려

ps ~~ -s N : 음악 검색 결과 중 N 번째 결과를 재생 또는 예약한다.
ex) ~ps 산들 취기를 빌려 -s 3

ls : 예약 리스트를 확인한다.

sh : 예약 리스트를 셔플하여 순서를 섞어준다.

p : 일시 정지
rs : 일시 정지된 음악을 다시 실행한다.

conn s : 음악을 플레이리스트에 저장한다.
ex) ~conn s 황치열 끝이라고 말할 것 같았어

conn l : 플레이리스트 목록 보기

conn d : 플레이리스트의 음악을 삭제한다.
ex) ~conn d (N)
리스트에서 N번 음악 삭제
N 생략시 마지막 음악 삭제
N = -1 : 모두 삭제

conn p : 플레이리스트를 재생 또는 예약한다.
